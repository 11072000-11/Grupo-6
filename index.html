Proyecto de Innovación - CONTADOR DE MONEDAS

C O I N B O X

El proyecto del contador de monedas, denominado "CoinBox", es una solución innovadora y atractiva diseñada para facilitar el proceso de conteo de monedas de forma automatizada. El objetivo principal es ofrecer una experiencia interactiva y divertida para los usuarios mientras cuentan sus monedas.

El CoinBox está diseñado en forma de una caja similar a un cajero automático, con una pantalla LCD integrada y un teclado numérico para una interfaz interactiva. Utiliza componentes electrónicos como un Arduino Nano y otros elementos para el funcionamiento del contador de monedas. El equipo Sinergia se enorgullece de presentar el CoinBox, un proyecto que combina la tecnología de Arduino Nano y otros componentes electrónicos para brindar un contador de monedas eficiente, preciso y atractivo en forma de cajero automático.

Proceso COINBOX

El proceso de conteo de monedas se inicia colocando las monedas en un compartimento específico en la parte superior de la caja. Una vez que las monedas son depositadas, el sistema de detección integrado en el CoinBox identifica y clasifica automáticamente las monedas por su denominación.

Se inició realizando un diseño 3D en Tinkercad con todas las piezas necesarias para nuestro producto. Posteriormente, hicimos un prototipo hecho de cartulina y cartón para tener noción de las piezas y sus dimensiones en tamaño real utilizadas para obtener el producto.

Prototipo


A continuación, las imágenes del proceso.

El modelo 3D en Tinkercad representa la parte del contador de monedas donde las monedas serán deslizadas y contabilizadas. Esta pieza ha sido diseñada con la intención de proporcionar un canal suave y seguro para que las monedas pasen sin obstrucciones.

La forma del modelo 3D es una estructura rectangular alargada, con una abertura en la parte superior para la entrada de las monedas. En la parte inferior, hay una abertura más pequeña que permite la salida controlada de las monedas hacia el mecanismo de conteo.

El diseño del modelo 3D ha sido optimizado para garantizar que las monedas pasen sin problemas a través del canal. Los bordes y esquinas han sido redondeados para evitar que las monedas se atasquen o se dañen durante su paso.

El modelo 3D representa el cajero automático diseñado para albergar el contador de monedas y proporcionar una experiencia interactiva para los usuarios. Este modelo está pensado para ser visualmente atractivo y funcional, simulando la apariencia de un cajero automático real.

El diseño del modelo 3D muestra una estructura rectangular con una pantalla LCD en la parte frontal para mostrar la información relevante al usuario. Debajo de la pantalla, se encuentra un teclado numérico para que los usuarios interactúen con el cajero automático.

Cartón

Utilizamos una hoja de papel para dibujar y diseñar el prototipo en cartulina. Definimos las dimensiones y las partes que formarán el cajero automático, como la pantalla LCD, el teclado numérico y el compartimento para las monedas.

Utilizamos una regla para medir y marcar las dimensiones de cada parte del prototipo en la cartulina. Luego, tijeras para recortar cada forma según las medidas marcadas. Se recortó con precisión para obtener piezas limpias y bien definidas.

Al tener las piezas recortadas, se comenzó a ensamblar el prototipo. Este prototipo es una representación visual del proyecto y no tiene funcionalidad electrónica real. Sin embargo, ayudará a visualizar y tener una idea de cómo se verá y se estructurará el CoinBox

Impresión 3D

La impresión 3D es un proceso de fabricación aditiva en el cual un objeto tridimensional se crea capa por capa a partir de un modelo digital. En el caso de CoinBox, la impresión 3D se utilizará para fabricar la pieza principal que conformarán el prototipo, y las demás piezas se realizarán en cartón.

A continuación, la descripción de algunos parámetros comunes en FlashPrint, un software de preparación y corte de modelos 3D, que consideramos al mandar a impresión la pieza.

Parámetros

Standard

El perfil standard se refiere a una configuración predeterminada que se utiliza para imprimir una amplia variedad de objetos con resultados satisfactorios. Configuración predeterminada que proporciona un equilibrio general entre velocidad de impresión, calidad y consumo de material.

Soportes

Soportes generados automáticamente por FlashPrint, se colocan en áreas donde el material necesita soporte para evitar que se caiga o se deforme durante la impresión. No requieren una configuración manual detallada, son ideales para modelos con geometrías complejas o salientes pronunciados.

Soportes

Los soportes tipo lineal proporcionan un buen equilibrio entre la necesidad de soporte y la facilidad de remoción. Al ser menos densos que los soportes sólidos, son más fáciles de quitar después de la impresión y dejan menos marcas o imperfecciones en la superficie del modelo.

Estas configuraciones son comunes en la impresión 3D y se utilizan para lograr un equilibrio entre calidad y adherencia. Una altura de capa de 0.18 mm proporciona una buena resolución y detalles finos en el modelo, mientras que un alto de primera capa de 0.27 mm ayuda a garantizar una correcta adherencia a la cama de impresión.

Cuando se establece una densidad de relleno del 15%, significa que aproximadamente el 15% del volumen total del objeto estará compuesto por material sólido en el interior. El resto del espacio se mantendrá como hueco o con una estructura en panal para ahorrar material y reducir el tiempo de impresión.

Al calentar la boquilla a 200º C, el filamento de PLA se funde y se extruye en forma de hilo fino que se deposita capa por capa para construir el objeto impreso. La temperatura de 200º C es crítica para asegurar que el filamento se derrita adecuadamente y fluya sin problemas a través de la boquilla sin obstrucciones.

El escalado uniforme en FlashPrint es una función que te permite cambiar el tamaño de un modelo 3D de manera proporcional en todas las dimensiones. Esta función conserva las proporciones originales del objeto y es útil cuando deseas redimensionar un modelo sin distorsionarlo.

Vídeo COINBOX

A lo largo de este proyecto, hemos combinado tecnología como Arduino Nano y componentes electrónicos con nuestra creatividad y habilidades para crear un prototipo funcional que simplifica y agiliza el conteo de monedas de manera divertida y efectiva. El CoinBox no solo cuenta las monedas, sino que también brinda una experiencia interactiva para los usuarios, ofreciendo una interfaz intuitiva y visualmente atractiva.
